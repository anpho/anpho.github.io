Infox

因为想要屏蔽到某些 Windows 11 的右键菜单项， Vibe 了一个用于导出 appxpackage 详细信息的 Powershell 脚本。
使用方法比较简单粗暴，下载后直接i.ps1 keyword就行，比如照片应用，可以运行i.ps1 Photos。

输出如：
